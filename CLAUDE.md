# claude-skills

A collection of general-purpose Claude Code skills for developer productivity and AI session optimisation. Skills here must work in any repo, any tech stack, with or without a plugin.

## Critical rules

- Skills must be prose-driven and general-purpose — no plugin- or framework-specific assumptions.
- Never delete content (from a repo's `CLAUDE.md`, `MEMORY.md`, or `openspec/INDEX.md`) without showing the user what's being removed and getting confirmation first.
- This repo has its own audit skill (`skills/optimise-claude-context/skill.md`) — apply its rules to this repo too.
